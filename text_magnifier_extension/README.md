# Text Magnifier

一个 Chrome / Edge 浏览器扩展，用来放大网页里的选中文本和输入框内容。

## 功能

- 当前标签页独立开关，互不影响。
- 选中普通网页文本时，弹出放大层。
- 点击 `input`、`textarea` 或可编辑区域时放大当前选区；没有选区时放大整个输入内容。
- 支持调整放大倍数和放大层宽度。
- 出于安全考虑，不会放大密码框内容。
- 放大层使用液态玻璃透镜风格：圆润边缘、背景模糊、蓝色高亮条、边缘高光和柔和折射感。
- 面板自带快捷操作：
  - 📌 **固定**：锁定当前位置，选新文本时只更新内容，面板不再跳动；再次点击解除。
  - 📋 **复制**：把当前放大文本一键复制到剪贴板。
  - **×**：关闭当前面板，按 `Esc` 也可关闭。
- **键盘快捷键**：默认 `Alt + Shift + M` 在当前标签页上一键切换开关；可在浏览器扩展快捷键设置中改键。
- 弹窗界面跟随系统的 `prefers-color-scheme` 自动适配明暗。

## 负载控制

- 开启后不持续监听鼠标移动，只在点击、选择、输入和窗口变化时更新。
- 滚动和缩放时会合并更新，避免高频重复计算。
- 放大层使用 CSS 视觉效果，不扫描页面全文，也不修改网页原始内容。

## 安装

1. 打开 Chrome 或 Edge 的扩展管理页面（`chrome://extensions` 或 `edge://extensions`）。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录（`text_magnifier_extension`）。

也可以直接下载仓库根目录下的 `text_magnifier_extension.zip`，解压后再加载。

> 修改键盘快捷键：在扩展管理页面左下角点击“键盘快捷键”（Chrome：`chrome://extensions/shortcuts`，Edge：`edge://extensions/shortcuts`）即可重新绑定。

## 使用

1. 打开任意普通网页。
2. 点击浏览器工具栏里的 `Text Magnifier`，或直接按快捷键。
3. 打开“当前标签页”开关。
4. 在网页中选中文本，或点击一个文本输入框。

按 `Esc` 可以关闭当前放大层；重新选择文本或输入内容后会再次显示。固定模式下面板不会随选区跳动，便于对照两段文本。
