# Text Magnifier

一个 Chrome / Edge 浏览器扩展，用来放大网页里的选中文本和输入框内容。

## 功能

- 当前标签页独立开关。
- 选中普通网页文本时，弹出放大层。
- 点击 `input`、`textarea` 或可编辑区域时，放大当前选区；没有选区时放大整个输入内容。
- 支持调整放大倍数和放大层宽度。
- 出于安全考虑，不会放大密码框内容。
- 放大层使用液态玻璃透镜风格：圆润边缘、背景模糊、蓝色高亮条、边缘高光和柔和折射感。

## 负载控制

- 开启后不持续监听鼠标移动，只在点击、选择、输入和窗口变化时更新。
- 滚动和缩放时会合并更新，避免高频重复计算。
- 放大层使用 CSS 视觉效果，不扫描页面全文，也不修改网页原始内容。

## 安装

1. 打开 Chrome 或 Edge 的扩展管理页面。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择这个目录：

   `/Users/max/Desktop/vs code/text_magnifier_extension`

## 使用

1. 打开任意普通网页。
2. 点击浏览器工具栏里的 `Text Magnifier`。
3. 打开“当前标签页”开关。
4. 在网页中选中文本，或点击一个文本输入框。

按 `Esc` 可以关闭当前放大层；重新选择文本或输入内容后会再次显示。
